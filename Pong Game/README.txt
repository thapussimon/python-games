This is a classic Pong game created using the turtle module in Python
Unlike the traditional pong game where opponent gets a point when it hits the boundary, I modified the program so that Only the player get a score when the ball hits the paddle.
The program is open for major improvements and contains minor bugs. 

###########################################################################################################################################


This program is uses basic python programming skills like creating functions, looping and some basic mathematics knowledge(Cartesian Plane).
The mathematics part is not complicated, Just play around with Numbers, until you get a clear picture about the game.
It would be a good start  to build this project after you have a basic understanding of python.
Basic Mathematics level algorithms are used to control the movement of the paddles, ball, collissions and even boundary checking.





#####################                       ABOUT  THE GAME      #########################################################################


Paddle A is at the LHS
Paddle B is at the RHS

Paddle A controls are "w" and "s" keys in the keyboard
Paddle B Controls are Top Arrow Key And Bottom Arrow Key
